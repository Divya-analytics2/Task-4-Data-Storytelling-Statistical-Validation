# 📊 Regional Sales Analysis & Hypothesis Testing (ANOVA)

## 📌 Project Overview
This project analyzes sales performance across four regions (West, East, Central, South) using Power BI and statistical hypothesis testing (One-Way ANOVA).

The objective is to determine whether there is a statistically significant difference in sales among regions.

## 📈 Hypothesis Testing

### Test Used:
One-Way ANOVA

### Significance Level:
α = 0.05

### Hypotheses:
- **H₀ (Null Hypothesis):** There is no significant difference in sales among regions.
- **H₁ (Alternative Hypothesis):** There is a significant difference in sales among regions.

### ANOVA Results:
- F-value = 0.8974
- P-value = 0.4416
- F-critical = 2.6058

### Decision:
Since P-value (0.4416) > 0.05, we fail to reject the null hypothesis.

### Conclusion:
There is no statistically significant difference in sales across regions.

---

## 🔎 Key Insights
- Sales are relatively balanced across all regions.
- Regional location does not significantly impact sales.
- Within-region variability suggests other influencing factors like product category and seasonality.

---

## 🚀 Business Recommendations
- Implement uniform marketing strategies across regions.
- Focus on product-level and seasonal analysis.
- Optimize promotions during low-performing months.
- Strengthen high-performing product categories.

---

## 📌 Project Outcome
The analysis concludes that regional differences do not significantly affect sales performance. Business strategies should focus on operational and product-based improvements rather than geographic factors.
